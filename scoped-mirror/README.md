# @100xpercent/mcp-server-pop-pay

MCP-convention alias for [`pop-pay`](https://www.npmjs.com/package/pop-pay). Identical functionality — this package exists only to match the `@scope/mcp-server-<name>` naming convention used by Anthropic's official MCP servers (e.g. `@modelcontextprotocol/server-filesystem`).

## Install

```bash
npm install -g @100xpercent/mcp-server-pop-pay
```

Exposes the same binaries as `pop-pay`: `pop-pay`, `pop-launch`, `pop-init-vault`, `pop-unlock`.

## Which should I use?

- **`pop-pay`** — primary package, recommended for most users
- **`@100xpercent/mcp-server-pop-pay`** — same code, scoped alias for MCP directory / convention alignment

Both track the same version and are published together on every release.

See the [main README](https://github.com/100xPercent/pop-pay#readme) for full documentation.
